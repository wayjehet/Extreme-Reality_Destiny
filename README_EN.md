About

Extreme Reality: Destiny

is a large-scale Minecraft modpack based on:

Minecraft 1.21.1 + NeoForge

Designed with a combination of:

- ⚔️ Adventure
- ⚙️ Technology
- 🏗️ Building
- 🌎 Exploration
- 🍲 Survival
- 📜 Quest Progression
- 🔫 Enhanced Combat

Created and maintained by:

Extreme Studio｜极致JiZhi工作室
